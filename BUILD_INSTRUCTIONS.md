# Инструкция по сборке и деплою MoreDota

## Вариант 1: Сборка для статического хостинга

### Шаг 1: Сборка проекта

Выполните в терминале:

```bash
cd /workspaces/default/code
pnpm install
pnpm run build
```

Это создаст папку `dist` с готовыми статическими файлами.

### Шаг 2: Настройка для виртуального хостинга

После сборки вам нужно:

1. Загрузить содержимое папки `dist` на ваш хостинг
2. Настроить `.htaccess` для правильной работы маршрутизации

Создайте файл `.htaccess` в корне сайта:

```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.html$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.html [L]
</IfModule>
```

### Шаг 3: Загрузка на хостинг

1. Подключитесь к хостингу через FTP/SFTP
2. Загрузите ВСЕ файлы из папки `dist` в корневую папку вашего сайта
3. Загрузите файл `.htaccess`

## Вариант 2: Использовать бесплатный специализированный хостинг

Это **рекомендуемый вариант** - проще и быстрее!

### Vercel (рекомендуется)

1. Зарегистрируйтесь на https://vercel.com
2. Установите Vercel CLI:
   ```bash
   npm install -g vercel
   ```
3. В папке проекта выполните:
   ```bash
   vercel
   ```
4. Следуйте инструкциям, привяжите домен moredota.com

**Преимущества:**
- Бесплатно
- Автоматический деплой при изменениях
- Очень быстрый CDN
- SSL-сертификат бесплатно
- Поддержка custom доменов

### Netlify

1. Зарегистрируйтесь на https://netlify.com
2. Перетащите папку проекта в их веб-интерфейс
3. Привяжите домен moredota.com

### Cloudflare Pages

1. Зарегистрируйтесь на https://pages.cloudflare.com
2. Подключите репозиторий или загрузите файлы
3. Настройте домен

## Вариант 3: Хостинг с поддержкой Node.js

Если ваш текущий хостинг поддерживает Node.js:

1. Загрузите все файлы проекта на хостинг
2. Установите зависимости: `pnpm install`
3. Соберите проект: `pnpm run build`
4. Настройте веб-сервер (nginx/Apache) для раздачи статических файлов из папки `dist`

---

## Рекомендация

Для MoreDota **лучше всего использовать Vercel** или **Netlify**:
- ✅ Бесплатно навсегда
- ✅ Автоматические обновления
- ✅ Высокая скорость
- ✅ SSL из коробки
- ✅ Можно привязать домен moredota.com

Обычный виртуальный хостинг (для PHP сайтов) не оптимален для React-приложений.
